CoherentStructures.jl

Tools for computing Lagrangian Coherent Structures in Julia

Introduction

CoherentStructures.jl is a toolbox for computing Lagrangian Coherent Structures in aperiodic flows in Julia. It has been developed in Oliver Junge's research group at TUM, Germany, by (in alphabetical order)

  • Alvaro de Diego (@adediego)
  • Daniel Karrasch (@dkarrasch)
  • Nathanael Schilling (@natschil)

Contributions from colleagues in the field are most welcome via raising issues or, even better, via pull requests.

Installation

First install the JuAFEM.jl package.

Then, run the following in the Julia REPL: ]add https://github.com/CoherentStructures/CoherentStructures.jl.git
